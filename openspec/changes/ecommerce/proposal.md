# E-commerce Features: Products, Cart & Mock Payment

## Context
The app is a React Native PWA (Expo + NativeWind) with a single splash screen. No product catalog, cart, or checkout flow exists.

## Problem
The app needs core e-commerce functionality to sell tobacco/accessory products as a PWA.

## Objectives
- Product listing screen with mock products (tabacaria-themed)
- Cart system with add/remove/quantity management
- Mock payment flow simulating credit card checkout
- No external dependencies beyond what's already installed
